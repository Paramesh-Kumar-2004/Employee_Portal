# Employee Portal

A web application for managing employee attendance.

## Features

* User authentication
* Employee attendance tracking
* Ability to mark attendance as present or absent
* Ability to save attendance records

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository
2. Open the index.html file in a web browser
3. Log in with a username and password
4. Click on the attendance button to mark attendance
5. Click on the save button to save the attendance record
